# Sentiment Analysis using Machine Learning

This project predicts whether a text review is Positive or Negative using Natural Language Processing and Machine Learning.

## Features
• Text preprocessing and tokenization  
• TF-IDF vectorization  
• Trained ML model for sentiment prediction  
• Ready-to-use saved models (.pkl / .joblib)

## Tech Stack
Python • Scikit-learn • Pandas • NumPy • NLP

## Project Files
- deeplearning_project.ipynb → Model training notebook
- sentiment_model.pkl → Trained model
- vectorizer.pkl → Text vectorizer
- tokenizer.pkl → Text tokenizer
